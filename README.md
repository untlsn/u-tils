# u-tils
Simple utils lib for JavaScript and TypeScript with lodash names

# Goals
* TypeScript first
* Tree-shaking
* No-deps
* Lodash names
* Compose and compose-l friendly
* Small chain api
* ES6+

# Chain - future
Inspired by the lodash chain, but much smaller with addons support
